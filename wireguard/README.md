# WireGuard

WireGuard provides the VPN tunnel between this VPS and Homelab Center.

## Configuration

- **Peer**: `homelab` at `10.13.13.2`
- **Allowed IPs**: `10.13.13.1`, `10.13.13.2`
- **Persistent keepalive**: Enabled for the homelab peer
- **Ports exposed**: 
  - `51820/udp` - WireGuard tunnel
  - `80`, `443` - Shared with Caddy via network namespace

## Network Namespace Sharing

Caddy and Authentik use `network_mode: "service:wireguard"` to share the WireGuard container's network namespace. This means they can access `127.0.0.1` within the WireGuard network context.

## Kernel Module

WireGuard requires the kernel module to be loaded before starting:

```sh
# Check if loaded
lsmod | grep wireguard

# Load if needed
sudo modprobe wireguard
```

## Generated Configuration

Configuration is auto-generated by the linuxserver/wireguard image in `./config/`. The peer configuration should be copied to Homelab Center.
